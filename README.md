# 🎨 Premium Theme for Pterodactyl

A beautiful dark theme with customizable colors for Pterodactyl Panel.

**By [volexion95-tech](https://github.com/volexion95-tech)**

## ✅ Compatible
- Pterodactyl 1.11.x / 1.12.x (Laravel 10/11)
- Node.js 20 / 22
- Debian 11/12, Ubuntu 20/22/24

## 🚀 Install
```bash
bash <(curl -s https://raw.githubusercontent.com/volexion95-tech/pterodactyl-premium-theme/main/install.sh)
```

## ❌ Uninstall
```bash
bash <(curl -s https://raw.githubusercontent.com/volexion95-tech/pterodactyl-premium-theme/main/uninstall.sh)
```

## 🎨 Change colors
Go to **Admin Panel → Theme** to customize colors and links.

After saving colors, rebuild:
```bash
cd /var/www/pterodactyl && yarn build:production
```
